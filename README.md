# Chaos Lab
visually simulate chaotic systems in rust.
e.g. Lorenz attractor, double pendulum, logistic map

## Setup

create a `.csv` in the cwd for logistic map data to write to. Set appropriate `path` varible in `main.rs`

### Python Plotting
in your terminal:
```
python3 -m venv venv

source venv/bin/activate // macos
venv\Scripts\activate.bat // Windows commandprompt

pip3 install matplotlib

```
## To Start
```
./scripts/start.sh
```
